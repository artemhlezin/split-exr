### Installation

- Download or clone this repo
`git clone https://github.com/artemhlezin/split-exr.git`

- add `split-exr` folder to your nuke plugin path

For example, modify init.py 
```python
import nuke

nuke.pluginAddPath("./python/split-exr")
```
